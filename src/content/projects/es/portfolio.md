## Descripción general

Este portafolio es una muestra de mi trabajo como desarrollador frontend. Está construido con Astro para generación estática, React para componentes interactivos y Tailwind CSS v4 para los estilos.

## Diseño

El diseño visual está inspirado en la estética cyberpunk y Blade Runner — lluvia animada, destellos de neón, horizonte urbano y superposición de líneas de escaneo. Todos los tokens de diseño se definen en CSS usando la configuración CSS-first de Tailwind v4.

## Internacionalización

El sitio es completamente bilingüe (inglés/español) usando el enrutamiento i18n integrado de Astro.
