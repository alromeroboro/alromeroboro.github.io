## Overview

This portfolio is a showcase of my work as a frontend developer. It is built with Astro for static generation, React for interactive components, and Tailwind CSS v4 for styling.

## Design

The visual design is inspired by cyberpunk and Blade Runner aesthetics — animated rain, neon glows, a city skyline, and scanline overlays. All design tokens are defined in CSS using Tailwind v4's CSS-first config.

## Internationalisation

The site is fully bilingual (English / Spanish) using Astro's built-in i18n routing.
